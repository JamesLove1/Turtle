To run the extension:

Extension Name: Turtles Genarator - Sofware to generate .ttl files

1. compile code e.g. 
>> Make extension_s 

2. run executable with place holder TTL file and .ttl out file e.g.
>> ./extension_s ./TTLs/turn.ttl ./out_Extension.ttl

3. Use instructions within program to operate the terminal

4. Check outputted TTL file (The file can be passed to the ./interp_s file).

Micilanous - important instructions worth reading
When using the rotate comand it is easy to press rotate one to many time.
Maybe try to press forwards to gain a sense for the direction you are 
heading post using rotate when mutiple rotate comands are used in close sucession.
